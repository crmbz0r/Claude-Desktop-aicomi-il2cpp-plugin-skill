# Claude Desktop aicomi-il2cpp-plugin skill
## A skill made for creating AiComi plugins within the Claude Desktop App [experimental]


- If you want to use it / try it out, either download the provided zip file from the releases or clone the repository if you want to make sure that it has the latest changes - if any are made after the zip release upload. If cloned, put "SKILL.md" and the "references" folder into a zip file.
- Then open https://claude.ai/customize/skills or your ***Claude Desktop Settings → Capabilities → Skills → Upload Skill*** (press the " **+** " icon at the top for upload button).
- Or in your new chat screen, click the " **+** " in the bottom left corner of the text input area, go to ***Skills → Managa Capabilities*** and then same procedure, press the " **+** " → **Upload Skill**
- Select your created zip file or the downloaded one from the [releases page](https://github.com/crmbz0r/Claude-Desktop-aicomi-il2cpp-plugin-skill/releases) (for example "aicomi-il2cpp-plugin.zip").
